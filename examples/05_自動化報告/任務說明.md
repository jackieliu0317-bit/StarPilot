## 💬 範例對話

### 情境 1：自動生成 Excel 報告

**你可以這樣說：**

```
我每週都要整理銷售數據做成 Excel 報告，有辦法自動化嗎？
資料在 data/sales.csv
```

**進階要求（可選）：**

```
報告要包含：
- 每個產品的總銷售額
- 前 5 名產品排行
- 銷售趨勢折線圖
- 各地區銷售圓餅圖
```

```
設定成定時執行，每週一早上 9 點自動生成
```

💡 **重點技巧**：
- 明確列出需要的統計項目
- 指定圖表類型
- 設定自動化排程

## 💡 提示

### 報告內容設計

✅ **完整報告應包含**：
1. 封面/標題頁
2. 目錄（多頁報告）
3. 執行摘要
4. 詳細分析（圖表+文字）
5. 結論與建議
6. 附錄（原始資料）

### 自動化報告的關鍵

1. **資料來源穩定**：確保檔案路徑和格式固定
2. **範本可重用**：建立標準報告範本
3. **錯誤處理**：資料缺失時的應對方式
4. **版本控制**：每次生成的報告要有版本識別
5. **通知機制**：完成後自動通知相關人員

### 圖表選擇建議

| 分析目的 | 推薦圖表 |
|----------|----------|
| 趨勢分析 | 折線圖 |
| 項目比較 | 長條圖 |
| 佔比分析 | 圓餅圖 |
| 關聯分析 | 散佈圖 |
| 層次結構 | 樹狀圖 |

## 🌟 小技巧

### 如何描述報告需求

**範例 1：明確的結構**
```
我要一份 Excel 報告，包含：
- 工作表1：原始資料
- 工作表2：統計摘要（平均、總計、排名）
- 工作表3：圖表（折線圖、長條圖、圓餅圖各一）
```

**範例 2：指定格式樣式**
```
PowerPoint 要用專業商務風格
- 藍色系配色
- 每頁標題用粗體
- 圖表用平面設計風格
```

### 自動化報告的最佳實踐

**命名規範**：
```
週報_2024-03-20.xlsx
月報_2024-03.pdf
Q1報告_2024.pptx
```

**資料夾結構**：
```
reports/
├── weekly/    # 週報
├── monthly/   # 月報
├── quarterly/ # 季報
└── archive/   # 歷史存檔
```

### 常見報告類型

| 報告類型 | 特點 |
|----------|------|
| 週報 | 簡短、重點式、即時性 |
| 月報 | 完整、詳細、包含趨勢 |
| 季報 | 綜合、策略性、前瞻性 |
| 專案報告 | 進度、問題、風險 |
| 績效報告 | KPI、達成率、分析 |
