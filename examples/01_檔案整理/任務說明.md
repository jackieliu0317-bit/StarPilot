# 📁 檔案整理任務

## 🎯 任務目標

學習如何用自然語言讓 AI 幫你整理檔案。

---

## 💬 與 AI 的對話範例

### 情境 1：基本檔案分類

```
👤：我想整理 Downloads 資料夾，把檔案按類型分類

🤖：好的，我會幫你建立一個檔案整理工具。請問你想怎麼分類？

👤：把 PDF 和 Word 放到「文件」資料夾
    把 JPG 和 PNG 放到「圖片」資料夾
    把 MP4 放到「影片」資料夾

🤖：了解，我會建立這個分類工具...
    [AI 會生成並執行程式碼]
    
    ✅ 完成！處理了 156 個檔案
       - 文件：42 個
       - 圖片：98 個  
       - 影片：16 個
```

---

### 情境 2：進階需求 - 加上日期分類

```
👤：可以再細分嗎？在每個類別下按照月份建立子資料夾

🤖：沒問題，我會修改成：
    文件/
      ├── 2024-11/
      ├── 2024-12/
    圖片/
      ├── 2024-11/
      └── 2024-12/
    
    要按照檔案的「建立日期」還是「修改日期」分類？

👤：修改日期

🤖：好的，已更新...
    [執行完成]
```

---

### 情境 3：安全確認

```
👤：等等，我想先看看會移動哪些檔案，不要真的移動

🤖：了解，我會先產生一個預覽清單...
    
    預計移動的檔案：
    📄 report.pdf → 文件/2024-12/
    📄 invoice.docx → 文件/2024-11/
    🖼️ photo1.jpg → 圖片/2024-12/
    ...
    
    確認要執行嗎？

👤：確認

🤖：開始移動檔案...
    ✅ 完成！
```

---

## 🎓 學到的技巧

1. **逐步細化需求**：先說大方向，再慢慢補充細節
2. **確認理解**：讓 AI 重述一次，確保理解正確
3. **安全第一**：可以先預覽再執行
4. **隨時修改**：發現需求變了就直接說

## 🚀 現在輪到你了！

開啟 Copilot Chat (`Ctrl+I` 或 `Cmd+I`)，試著對話：

### 練習 1：基礎整理
```
「幫我整理桌面的檔案，把不同類型分到不同資料夾」
```

### 練習 2：批次重新命名
```
「我有 100 張照片，想重新命名成：
 旅遊_001.jpg
 旅遊_002.jpg
 這樣的格式」
```

### 練習 3：清理舊檔案
```
「找出超過 30 天沒用過的檔案，移到 Archive 資料夾」
```

## 💡 提示

- 不確定怎麼描述？就用最直白的方式說
- AI 不理解？換個說法再試一次
- 想修改？直接說「改成...」就好
- 記住：**你不需要看懂程式碼，只要確認結果對不對！**

## 📊 Excel 進階範例

我們準備了一個 Excel 檔案 `測試資料_待整理/attend.xlsx`，包含：
- **員工名單**：含敏感個資（姓名、身分證字號），共 30 筆
- **計算0901 ~ 計算1028**：20 個「計算」開頭的工作表，每日出勤計算資料

試試以下兩個進階任務！

---

### 範例 4：敏感資料匿名化

**你可以這樣說：**

```
幫我處理 attend.xlsx 檔案，把所有工作表中的敏感資料做匿名化處理：
- 姓名改成亂數生成的假名
- 身分證字號改成亂數生成的假號碼
要保持資料的關聯性（同一個人在不同工作表要對應相同的假資料）
```

**進階要求（可選）：**
```
幫我產生一份對照表，以便日後還原
```

💡 **重點技巧**：
- 要求「保持關聯性」確保資料一致性
- 可以要求產生對照表以便日後還原
- 敏感資料處理前先備份原檔！

---

### 範例 5：合併多個工作表

**你可以這樣說：**

```
幫我把 attend.xlsx 裡所有「計算」開頭的工作表合併成一個
```

**進階要求（可選）：**
```
垂直合併，加上「來源工作表」欄位來標註資料來自哪個工作表
```

```
把合併後的資料也匯出成 CSV
```

💡 **重點技巧**：
- 用「○○開頭」來描述篩選條件更直覺
- 合併時加入來源標記能追蹤資料出處
- CSV 匯出時注意編碼問題（UTF-8 with BOM）

---

## 🔥 挑戰任務

結合以上技巧，嘗試一次完成多個步驟：

```
先把所有「計算」工作表合併，然後對合併後的資料做敏感資料匿名化，
最後匯出成一個乾淨的 CSV 檔案
```

這就是 AI 輔助資料處理的威力 — 用自然語言描述複雜的多步驟任務！
