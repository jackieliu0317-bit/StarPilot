## 💡 提示

### 資料分析的對話技巧

✅ **好的描述**：
```
分析 data/sales.csv 檔案，計算每個產品的平均銷售額，並用長條圖顯示前 10 名
```

❌ **不好的描述**：
```
分析資料（什麼資料？分析什麼？）
```

### 常用分析類型

| 分析類型 | 說明 | 適合用途 |
|----------|------|----------|
| 統計摘要 | 平均值、中位數、標準差 | 了解資料分布 |
| 排名分析 | Top N、Bottom N | 找出最佳/最差項目 |
| 趨勢分析 | 時間序列、成長率 | 觀察變化趨勢 |
| 分布分析 | 直方圖、圓餅圖 | 了解比例分配 |
| 關聯分析 | 散佈圖、相關係數 | 找出變數關係 |

### 圖表選擇建議

- **比較數值** → 長條圖 (Bar Chart)
- **顯示趨勢** → 折線圖 (Line Chart)
- **比例分配** → 圓餅圖 (Pie Chart)
- **分布情況** → 直方圖 (Histogram)
- **多維關係** → 散佈圖 (Scatter Plot)
