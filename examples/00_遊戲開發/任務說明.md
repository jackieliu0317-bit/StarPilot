# 🎮 遊戲開發任務

## 🎯 任務目標

學習如何用自然語言讓 AI 幫你建立經典遊戲，體驗「從想法到可玩遊戲」的完整過程。

---

## 🐍 專案 1：貪食蛇遊戲

### 第一步：建立基本遊戲

**你可以這樣說：**

```
我想做一個貪食蛇遊戲，用網頁版本
```

**進階要求（可選）：**

```
使用 HTML/JavaScript 建立，用 Canvas 繪製遊戲畫面
```

```
蛇用方向鍵控制，吃到食物會變長，撞到自己或牆壁會結束
```

---

### 🌐 如何執行網頁版遊戲

**你可以這樣說：**

```
幫我啟動本地網頁伺服器來測試這個遊戲，檔案在 playground 資料夾
```

**手動執行指令**：
```bash
cd /workspaces/StarPilot/playground && python3 -m http.server 8000 --bind 0.0.0.0
```

> 💡 **提示**：在 Codespaces 中，Port 會自動轉發，你可以直接在瀏覽器中開啟遊戲！

---

### 第二步：增加功能

**你可以這樣說：**

```
可以加上計分板和速度提升嗎？每吃 5 個食物速度變快一點
```

```
再加上暫停功能，按空白鍵可以暫停/繼續遊戲
```

```
加上最高分紀錄功能，會儲存在本地
```

---

### 第三步：美化遊戲

**你可以這樣說：**

```
可以讓畫面更好看嗎？
- 蛇的顏色改成綠色漸層
- 背景改成深色
- 加上網格線
```

```
改成像素風格，8-bit 復古感覺
```

## 🧱 專案 2：俄羅斯方塊

### 第一步：建立遊戲

**你可以這樣說：**

```
我想做俄羅斯方塊遊戲，使用經典配色
```

**進階要求（可選）：**

```
包含 7 種不同形狀的方塊，方向鍵左右移動、向下加速，空白鍵旋轉方塊
```

---

### 第二步：調整難度

**你可以這樣說：**

```
遊戲太簡單了，可以加快速度嗎？
- 提高初始速度 30%
- 每消除 3 行就提升速度
```

---

### 第三步：加入進階功能

**你可以這樣說：**

```
可以加上：
1. 下一個方塊預覽（顯示在右側面板）
2. 最高分紀錄（儲存在本地檔案）
3. 遊戲結束時顯示排行榜
```

## 🎓 你會學到什麼

### 1. 遊戲開發概念
- **不需要懂遊戲引擎**，只要描述你想要的效果
- **不需要懂圖形介面**，AI 會處理所有視覺化
- **不需要懂物理計算**，AI 會實作碰撞偵測

### 2. 與 AI 的迭代開發
- 先做出基本版本
- 逐步加入新功能
- 根據試玩體驗調整

### 3. 需求描述技巧

✅ **好的描述**：
```
我想做貪食蛇，蛇頭用深綠色，食物用紅色圓圈，背景是黑色，視窗大小 600x600
```

❌ **不好的描述**：
```
做一個遊戲
```

## 🚀 現在開始

### 快速開始建議

1. **先選一個遊戲**
   - 貪食蛇（最簡單，推薦新手）
   - 俄羅斯方塊（稍微複雜，有成就感）

2. **開啟 Copilot Chat** (`Ctrl+I` 或 `Cmd+I`)

3. **對 AI 說**：
   ```
   我想做一個 [遊戲名稱] 遊戲，請幫我建立一個可以玩的版本
   ```

4. **試玩並改進**
   - 玩玩看遊戲
   - 想想可以改進的地方
   - 告訴 AI 你的想法

5. **分享成果**
   - 把你的遊戲截圖或錄影
   - 在 `playground/` 資料夾保存
   - 分享給朋友一起玩！
