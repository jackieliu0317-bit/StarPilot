## 🐍 專案 1：貪食蛇遊戲

### 💬 與 AI 的對話範例

#### 第一步：建立基本遊戲

```
👤：我想做一個貪食蛇遊戲

🤖：好的！我會幫你建立一個貪食蛇遊戲。請問你想要：
    1. 視窗版本（用 pygame）
    2. 網頁版本（用 HTML/JavaScript）
    3. 終端機版本（純文字）

👤：網頁版本

🤖：了解，我會用 HTML/JavaScript 建立一個網頁版貪食蛇遊戲。
    遊戲會包含：
    - 使用 Canvas 繪製遊戲畫面
    - 蛇的移動（方向鍵控制）
    - 食物的生成
    - 吃到食物會變長
    - 撞到自己或牆壁會結束
    
    正在生成程式碼...
    [生成完成] snake_game.html
    
    要在瀏覽器中開啟看看嗎？

👤：開啟

🤖：[瀏覽器自動開啟遊戲頁面]
    ✅ 遊戲啟動成功！
    - 使用方向鍵控制蛇的移動
    - 吃到紅色食物會加分
    - 可以直接分享這個 HTML 檔案給朋友玩
```

---

#### 🌐 如何執行網頁版遊戲

**方法一：請 Copilot 幫你啟動（最簡單）**

```
👤：幫我啟動本地網頁伺服器來測試這個遊戲
    檔案在 playground 資料夾

🤖：好的，我會啟動本地伺服器...
    [執行中] http://localhost:8000
    
    ✅ 伺服器已啟動！
    點擊上方的連結就能在瀏覽器中玩遊戲
```

**方法二：手動執行指令**

在終端機中執行：
```bash
cd /workspaces/StarPilot/playground && python3 -m http.server 8000 --bind 0.0.0.0
```

然後：
1. VS Code 會自動偵測到 Port 8000
2. 右下角會彈出通知，點擊「Open in Browser」
3. 或者點擊「PORTS」分頁，找到 8000，點擊地球圖示

> 💡 **提示**：在 Codespaces 中，Port 會自動轉發，你可以直接在瀏覽器中開啟遊戲！

## 🎓 你會學到什麼

### 1. 遊戲開發概念
- **不需要懂遊戲引擎**，只要描述你想要的效果
- **不需要懂圖形介面**，AI 會處理所有視覺化
- **不需要懂物理計算**，AI 會實作碰撞偵測

### 2. 與 AI 的迭代開發
- 先做出基本版本
- 逐步加入新功能
- 根據試玩體驗調整

### 3. 需求描述技巧
```
好的描述：
「我想做貪食蛇，蛇頭用深綠色，食物用紅色圓圈，
 背景是黑色，視窗大小 600x600」

不好的描述：
「做一個遊戲」
```

## 🚀 現在開始

### 快速開始建議

1. **先選一個遊戲**
   - 貪食蛇（最簡單，推薦新手）
   - 俄羅斯方塊（稍微複雜，有成就感）

2. **開啟 Copilot Chat** (`Ctrl+I` 或 `Cmd+I`)

3. **對 AI 說**：
   ```
   我想做一個 [遊戲名稱] 遊戲
   請幫我建立一個可以玩的版本
   ```

4. **試玩並改進**
   - 玩玩看遊戲
   - 想想可以改進的地方
   - 告訴 AI 你的想法

5. **分享成果**
   - 把你的遊戲截圖或錄影
   - 在 `playground/` 資料夾保存
   - 分享給朋友一起玩！

## 💡 實用技巧

```
👤：遊戲速度太快了，可以慢一點嗎？

```

