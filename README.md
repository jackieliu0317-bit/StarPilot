## 🚀 快速開始（5 分鐘上手）

### 步驟一：Fork 這個專案

1. 登入你的 GitHub 帳號
2. 點擊本頁面右上角的 **Fork** 按鈕
3. 這會將專案複製一份到你的帳號下（你可以隨意修改，不會影響原始版本）

### 步驟二：啟動 Codespaces

1. 在你的 Fork 專案頁面，點擊綠色的 **<> Code** 按鈕
2. 切換到 **Codespaces** 頁籤
3. 點擊 **Create codespace on main**
4. 等待 1-2 分鐘，瀏覽器會自動開啟 VS Code 線上環境

> 💡 **魔法時刻**：環境會自動安裝 Python、所有套件、以及 GitHub Copilot 擴充功能。你什麼都不用做！

### 步驟三：開始你的第一支程式

1. 點擊左側檔案列表中的 `01_暖身_與AI的第一支舞.py`
2. 按照檔案內的註解指示，開始你的 AI 詠唱之旅
3. **記住**：輸入中文註解後，按 `Tab` 鍵接受 Copilot 的建議

## 🧙 Copilot 詠唱指南（Prompt 小卡）

### ✅ 好的註解（具體、明確、有脈絡）

```python
# 讀取 data 資料夾中的 範例銷售數據.xlsx,將資料存入 df 變數

# 篩選出「銷售額」大於 10000 的資料列,並依照「日期」排序

# 計算每個「產品」的平均單價,用長條圖顯示前 5 名
```

### ❌ 不好的註解（模糊、太簡略、缺乏情境）

```python
# 讀取資料        <-- 什麼資料?從哪裡讀?
# 處理一下        <-- 要處理什麼?怎麼處理?
# 畫圖           <-- 畫什麼圖?用什麼資料?
```

### 🎯 黃金法則：三個要素

1. **動作**：要做什麼?（讀取、計算、篩選、畫圖）
2. **對象**：針對什麼資料?（欄位名稱、檔案路徑）
3. **條件**：有什麼限制?（大於某數值、前幾名、排序方式）

## 💡 常見問題 (FAQ)

### Q1: Copilot 沒有自動跳出建議?

**A**: 請確認：
1. 左下角是否顯示 Copilot 圖示（已啟用）
2. 註解結尾是否按下 `Enter` 並稍等 1-2 秒
3. 嘗試按 `Ctrl + Enter` 手動觸發建議面板

### Q2: 生成的程式碼有錯誤?

**A**: 這很正常！AI 不是 100% 完美。請：
1. 看錯誤訊息的關鍵字（通常在最後一行）
2. 在錯誤行上方寫註解：`# 修正上方的錯誤`
3. 或直接問 Copilot Chat：「為什麼會出現 XXX 錯誤?」

### Q3: 我可以用自己的資料嗎?

**A**: 當然！請將你的 Excel/CSV 檔案放到 `data/` 資料夾,然後在程式碼中修改檔案路徑即可。

### Q4: Codespaces 會過期嗎?

**A**: 
- 免費額度：每月 60 小時（對這堂課絕對夠用）
- 30 天未使用會自動刪除,但你的程式碼會保留在 GitHub 上
- 隨時可以重新啟動
